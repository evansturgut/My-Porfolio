## 📁 Repository Structure & Code Overview

This project is built using semantic, accessible web technologies structured to optimize load speeds and modern design principles.

```plaintext
├── index.html      # Main markup showcasing experience, skills, projects, and contact info
├── style.css       # Dynamic stylesheets managing Light/Dark mode and cross-breakpoint styling
└── script.js      # Enhanced client-side interactions, scroll-animations, and form verification
Core Web Features
Adaptive Dark Mode: Uses native CSS Custom Variables (:root / [data-theme="dark"]) synced to browser local storage via lightweight JavaScript execution.

Dynamic Typing Effect: An interactive header animation that introduces my core competencies asynchronously.

Scroll-Spy Navigation: Tracks element bounding rectangles relative to viewport scrolls to seamlessly update interactive header classes.

Form Validation Engine: A pure JS client-side data handler to screen, trim, and parse user messages before structural submission.
